# Code of Conduct

This project is committed to a friendly, respectful, and constructive community.

## Our Standard

Please be respectful in issues, pull requests, reviews, and discussions.

Examples of encouraged behavior:

- being patient and welcoming
- giving actionable, specific feedback
- assuming good intent while still being honest about problems
- keeping criticism focused on the code, logs, or behavior rather than the person

Examples of unacceptable behavior:

- harassment, insults, or personal attacks
- discriminatory or hateful language
- intentionally hostile or derailing comments
- publishing someone else's private information without permission

## Enforcement

Maintainers may remove, edit, or reject comments, issues, pull requests, code, or other contributions that do not follow this code of conduct.

If you experience or witness unacceptable behavior, please contact the maintainer privately instead of escalating in public.

## Scope

This code of conduct applies to repository discussions, issues, pull requests, and other project spaces.
